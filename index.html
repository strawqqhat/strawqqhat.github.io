关于我

✨ 大家好，我是 strawqqhat，一个写了 10 年代码但头发依然健在的 Java 程序员！

江湖人称 "码农界的咖啡因战士"，擅长用 System.out.println("Hello World"); 开启美好的一天，再用 try-catch 接住生活的暴击。过去十年，我和 Java 从 "初恋" 到 "老夫老妻"，经历了 Spring 全家桶 的洗礼，也曾在 JVM 调优 的深海里反复横跳。

我的日常：写 Bug → 修 Bug → 假装没写过 Bug，但偶尔也会写出让同事直呼 "这代码居然能跑？" 的神奇逻辑。热爱 开源社区，梦想是某天能写出一款让全世界程序员少加班的框架（目前还在梦里）。

技术信仰："代码可以重构，但咖啡不能停！" 🚀☕

（P.S. 如果你发现这段自我介绍有 Bug，欢迎 PR，但请温柔一点，毕竟我是个 "资深" 但依然会犯低级错误的码农 😂）
