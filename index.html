✨ 大家好，我是 strawqqhat，一个写了 10 年代码但头发依然健在的 Java 程序员！

我的日常：写 Bug → 修 Bug → 假装没写过 Bug，但偶尔也会写出让同事直呼 "这代码居然能跑？" 的神奇逻辑。热爱 开源社区，梦想是某天能写出一款让全世界程序员少加班的框架（目前还在梦里）。
